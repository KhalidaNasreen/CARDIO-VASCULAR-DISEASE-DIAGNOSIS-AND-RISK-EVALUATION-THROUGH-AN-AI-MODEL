# Cardiovascular Disease Diagnosis and Risk Evaluation

This project focuses on diagnosing cardiovascular diseases and evaluating patient risk using deep learning–based medical image analysis with a Mamba-based Voxel Morph model.

## Features
- Medical image–based cardiovascular disease analysis
- Deep learning model for risk evaluation
- Voxel-based morphological feature extraction
- Supports early-stage disease diagnosis
- Research-oriented healthcare solution

## Technologies Used
- Python
- Deep Learning
- Mamba-Based Voxel Morph Model
- NumPy
- Pandas
- Medical Image Processing

## Project Structure
- model/ : trained deep learning model
- templates/ : user interface files (if applicable)
- test_data/ : sample medical imaging data
- app.py : main application file

## How to Run
1. Install required libraries
2. Run `python app.py`
3. Access the application via local server
